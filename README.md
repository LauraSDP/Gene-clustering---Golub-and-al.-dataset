# Gene-clustering-Golub-and-al.-dataset

This repository contains a Python notebook implementing clustering algorithms and predictive models on a gene expression dataset from DNA microarrays. The goal is (1) to identify distinct gene expression patterns between patients suffering from acute myeloid leukemia or acute lymphoid leukemia and (2) to classify them according to their cancer type.
The dataset consists of DNA microarray data and is from Golub and al., Molecular Classification of Cancer: Class Discovery and Class Prediction by Gene Expression, Science 286:531-537. (1999). Published: 1999.10.14. 
The notebook was originally developed and published on Kaggle.

## Structure 
- 'gene-clustering-golub-and-al-dataset': notebook containing the analysis of the gene expression profile of both cancer types and the ML models
